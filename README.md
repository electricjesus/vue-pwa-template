# k

client provisioning tool.

### getting started

#### development

npm only:

    npm install
    npm run dev

or with [yarn](https://yarnpkg.com/):
    
    npm install -g yarn
    yarn
    npm run dev

#### production

  npm run build
    docker build -t mewe/k .
    docker-compose up
